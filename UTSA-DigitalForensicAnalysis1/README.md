# Digital Forensics Analysis I - UTSA

## Course Overview

This course introduced the fundamental data structures, software tools, and forensic techniques used to locate and recover digital evidence. My work focused on computer file systems, forensic imaging, evidence examination, file recovery, and the proper handling of digital evidence.

## Skills Developed

- Forensic imaging and evidence collection
- File-system analysis
- Deleted-file recovery
- Metadata and timestamp examination
- Timeline reconstruction
- Evidence integrity and chain of custody
- Documentation of forensic findings

## Tools Used

- FTK Imager
- Autopsy
- Sleuth Kit
- WinHex
- Linux command-line tools
- Visual Studio Code
