# Information Assurance and Security – UTSA

## Course Overview

This course introduced the fundamental principles of information assurance and cybersecurity. My work focused on understanding common threats to information systems, evaluating security risks, and examining the technologies and strategies used to protect data, networks, and organizational systems.

The course covered fraud, eavesdropping, network traffic analysis, intrusion detection and prevention, hacking, malware, cryptography, risk management, and secure system architectures.

## Skills Developed

- Information assurance principles
- Identification of cybersecurity threats
- Network traffic analysis
- Intrusion detection and prevention
- Malware and virus analysis concepts
- Cryptographic security principles
- Security risk identification
- Risk assessment and management
- Secure architecture fundamentals
- Evaluation of security controls
- Protection of data confidentiality, integrity, and availability

## Core Topics

- Fraud and cybercrime
- Eavesdropping and traffic analysis
- Intrusion detection and prevention systems
- Hacking techniques and defensive security
- Viruses and other forms of malware
- Encryption and cryptography
- Risk management
- Secure network and system architectures
