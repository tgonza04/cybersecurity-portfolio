# Intrusion Detection and Incident Response – UTSA

## Course Overview

This course provided hands-on experience with the technologies, methodologies, and procedures used to detect and respond to cybersecurity incidents. My work focused on identifying suspicious network activity, examining intrusion evidence, analyzing security events, and following proper incident response and forensic procedures.

The course also examined cybercrime laws, intellectual property issues, evidence handling, and the role of digital forensics in identifying and tracking intruders.

## Skills Developed

- Network intrusion detection
- Network traffic and packet analysis
- Security log examination
- Incident identification and classification
- Incident containment and response
- Network forensic investigation
- Evidence preservation and documentation
- Identification of indicators of compromise
- Intruder tracking and attack reconstruction
- Understanding of cybercrime laws and procedures

## Tools Used

- Snort
- Wireshark
- tcpdump
- NetworkMiner
- Nmap
- Windows and Linux virtual machines
- Command-line analysis tools
